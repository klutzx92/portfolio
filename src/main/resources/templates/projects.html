PrairieLearn

Online Exam Creation Tool

Semester project for my Software Engineering Team Project lab course. PrairieLearn is an open-source exam creation tool developed by the UC Berkeley ACE Labs. Our small Agile team of five members were tasked with developing a user interface app through which instructors could connect to their repos through GitHub APIs, pull exam questions, input configurations, and export as a JSON file. Prior to our app, instructors had to hard code each assessment. App built using Ruby on Rails, HTML/CSS/Javascript.

GITLET

Mini Form of the Version-Control System Git

Project for my Data Structures course at UC Berkeley. Fall 2019. All code was written from scratch using Java. I designed and implemented a version-control system that mimics some of the basic features of Git. Commands include init, add, commit, rm, log, global-log, find, status, checkout, branch, rm-branch, reset, and merge. I am super proud of this project as it forced me to think carefully about my design and use of various data structures. This was also the first project I ever wrote from scratch that had a great deal of complexity.
